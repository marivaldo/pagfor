# Pagfor

Implementação do Layout PAG-FOR Bradesco para pagamento de Fornecedores.
Com base na documentação MP_4008-0096-00, Data: 06/08/2010

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'pagfor'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pagfor
